# Chili-Peppers-Zeaxanthin
A multimodal feature fusion model (MCSF) integrating spectral and chemical data for high-accuracy prediction of zeaxanthin content in peppers, achieving R² up to 0.93 and RPD up to 4.14.
The code for the core model and main experimental results is provided. Ablation studies, comparative experiments, and sensitivity analysis involve numerous variant configurations and are not included for brevity; they can be made available upon reasonable request.
